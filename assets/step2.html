Step 2: Download and install the SDKs and Tools

  • Install Eclipse if you don't have it already.
  • Set your Java Compliance Level to Java 1.6: Eclipse->Preferences->Java->Compiler->Compiler Compliance Level->1.6
  • Install Android SDK and the Eclipse Plugin
  • Install Git: Win Setup, OSX Setup, Linux Setup
  • Install Facebook SDK by cloning the GitHub repository:
    • git clone git://github.com/facebook/facebook-android-sdk.git
  • For Emulator testing, create Virtual devices by going to Eclipse->Window->Android SDK and AVD Manager->Virtual devices

Notes: This is basically the routine you will perform if you want to write any Android application. For those who have already set up the Android development environment, all you have to do is go to GitHub and download the Facebook SDK.
